Twig Extensions
===============

.. toctree::
    :hidden:

    debug
    text
    i18n

The Twig Extensions repository provides several useful extensions for Twig:

* *:doc:`Twig_Extensions_Extension_Debug <debug>`*: Provides tags and filters
  to ease template debugging;

* *:doc:`Twig_Extensions_Extension_Text <text>`*: Provides useful filters for
  text manipulation;

* *:doc:`Twig_Extensions_Extension_I18n <i18n>`*: Adds internationalization
  support via the ``gettext`` library.
